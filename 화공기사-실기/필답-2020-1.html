McCabe-Thiele법에서 최소이론단수를 구하기 위한 공급선의 방정식에서 q의 식과 의미를 쓰시오.
q = 1 : 주어진 원료가 끓는점의 순수 액상 (포화액체)
q > 1: 원료가 끓는점 미만의 액체
0 < q < 1 : 액상과 기상이 균형을 이룬 상평형 상태
q = 0 : 원료가 끓는점의 포화 기체
q < 0 : 원료가 끓는점을 넘은 기체
1) q = 1 : saturated-liquid (q가 1이면 L'=L+F, 즉 모두 액체로 내려간다.)
2) q = 0 : saturated-vapor (q가 0이면 L'=L, Feed는 모두 기체로 올라간다.)
3) 0 < q < 1 : partially flashed
4) q > 1 : subcooled-liquid (Feed가 오히려 Vapor중 일부를 Liquid로 끌어내린다.)
5) q < 0 : superheated-vapor (Feed가 오히려 Liquid중 일부를 Vapor로 올려보낸다.)

$ H = \pu{ 5.19E7 atm } \times \dfrac{\pu{ 760 mmHg }}{\pu{ 1 atm }} = \pu{ 3.9444E16 mmHg } \\ n_\ce{H2O} = \pu{ 100 kg } \times \dfrac{\pu{ 1 kmol }}{\pu{ 18 kg }} = \pu{ \dfrac{ 50 }{ 9 } kmol } \\ P_\ce{H2} = H\cdot x_\ce{H2} \Longrightarrow x_\ce{H2} = \dfrac{\pu{ 200 mmHg }}{\pu{ 3.9444E10 mmHg }} = \pu{ 5.0705E-9 } \\ \begin{align*} x_\ce{H2} &= \dfrac{ n_\ce{H2} }{ n_\ce{H2O} + n_\ce{H2}} (\text{분모의 } n_\ce{H2} \text{는 } n_\ce{H2O} \text{에 비해 너무 작으므로 무시 가능하다.}) \\ \Longrightarrow \pu{ 5.0705E-9 } &= \dfrac{ n_\ce{H2} }{ 50/9\ \pu{ kmol } } \\ \end{align*} \\ \therefore n_\ce{H2} = \pu{ 2.8169E-8 kmol } \\ \Longrightarrow w_\ce{H2} = \pu{ 2.8169E-8 kmol } \times \pu{2.01588 kg/kmol} = \pu{5.6785E-8 kg} $

Soret effect(Thermo Diffusion)와 Dufour effect(Diffusion Thermal)에 대해서 설명하시오.
Soret effect(Thermo Diffusion)는 액체 혼합물 중에 온도 기울기가 있으면 물질의 확산이 일어나서 농도 기울기가 생기는 현상.
Dufour effect(Diffusion Thermal)는 액체 혼합물 중에 농도 기울기가 있어 물질이 서로 확산하면 온도 기울기가 생기는 현상. Soret effect(Thermo Diffusion)의 반대 개념.
