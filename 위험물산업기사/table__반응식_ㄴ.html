나트륨 + 에탄올 $$\ce{2 \underset{ \text{ 나트륨 }}{ Na } + 2 \underset{ \text{ 에탄올 }}{ C2H5OH } → 2C2H5ONa + \underset{ \text{ 수소 }}{ H2 } }$$
나트륨 + 이산화탄소 $$\ce{4 \underset{ \text{ 나트륨 }}{ Na } + 3 \underset{ \text{ 이산화탄소 }}{ CO2 } → 2 \underset{ \text{ 탄산나트륨 }}{ Na2CO3 } + C }$$

니트로벤젠 제조 $$\ce{C6H6 + HNO3 →[C-H2SO4][니트로화] C6H5NO2 + \underset{ \text{ 물 }}{ H2O } }$$

마그네슘 + 황산 $$\ce{ Mg + \underset{ \text{ 황산 }}{ H2SO4 } → MgSO4 + H2}$$

메탄올 (메틸알코올) 연소식 $$\ce{2CH3OH + 3 \underset{\text{산소}}{ O2 } → 2 \underset{\text{이산화탄소}}{ CO2 } + 4 \underset{ \text{ 물 }}{ H2O } }$$
