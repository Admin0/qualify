분말소화약제의 반응식
종별 주성분 색상 적응화재 반응식
제1종분말 중탄산나트륨($\ce{ NaHCO3 }$) 백색 BC 270℃ $$\ce{ 2NaHCO3 -> Na2CO3 + CO2 + H2O }$$
850℃ $$\ce{ 2Na2CO3 -> Na2O + 2CO2 + H2O }$$
제2종분말 중탄산칼륨($\ce{ KHCO3 }$) 보라색 BC $$\ce{ 2KHCO3 -> K2CO3 + CO2 + H2O }$$
제3종분말 인산암모늄($\ce{ NK4H2PO4 }$) 담홍색 ABC 1차 $$\ce{ NH4H2PO4 -> H3PO4 + NH3 }$$
2차 $$\ce{ NH4H4PO4 -> HPO3 + NH3 + H2O }$$
기타 소화약제의 반응식
화학포 $$\ce{ 6NaHCO3 + Al2(SO4)3*18H2O → 3Na2SO4 + 2Al(OH)3 + 6CO2 + 18H2O }$$
강화액 $$\ce{ K2CO3 + H2SO4 + H2O → K2SO4 + 2H2O + CO2↑ }$$
강/알칼리 $$\ce{ 2NaHCO3 + H2SO4 → Na2SO4 + 2CO2 + 2H2O }$$
