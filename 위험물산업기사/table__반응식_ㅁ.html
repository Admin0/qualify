반응식 (ㅁ)
반응 반응식
마그네슘 + 물 $$\ce{ Mg + 2 \underset{ \text{ 물 }}{ H2O } → Mg(OH)2 + H2}$$
마그네슘 + 염산 $$\ce{ Mg + 2 \underset{ \text{ 염산 }}{ HCl } → MgCl2 + H2}$$
마그네슘 + 이산화탄소 $$\ce{2Mg + \underset{ \text{ 이산화탄소 }}{ CO2 } → 2MgO + C }$$
마그네슘 + 황산 $$\ce{ Mg + \underset{ \text{ 황산 }}{ H2SO4 } → MgSO4 + H2}$$
마그네슘 연소식 $$\ce{ 2Mg + \underset{\text{산소}}{O2} → 2MgO}$$
메탄 분해식 $$\ce{CH4 + 2 \underset{\text{산소}}{ O2 } → \underset{ \text{ 이산화탄소 }}{ CO2 } + 2 \underset{ \text{ 물 }}{ H2O } }$$
메탄올 (메틸알코올) 연소식 $$\ce{2CH3OH + 3 \underset{\text{산소}}{ O2 } → 2 \underset{\text{이산화탄소}}{ CO2 } + 4 \underset{ \text{ 물 }}{ H2O } }$$
메틸리튬 + 물 $$\ce{ (CH3)Li + \underset{ \text{ 물 }}{ H2O } → LiOH + CH4}$$
