반응식 (ㅅ)
반응 반응식
삼산화크롬 + 물 $$\ce{ CrO3 + \underset{ \text{ 물 }}{ H2O } → H2CrO4}$$
삼산화크롬 열분해식 $$\ce{ 4CrO3 → 2Cr2O3 + 3O2}$$
삼황화린 연소식 $$\ce{ P4S3 + 8O2 → 2P2O5 + 3SO2}$$
수소화알루미늄리튬 + 물 $$\ce{LiAlH4 + 4 \underset{ \text{ 물 }}{ H2O } → LiOH + Al(OH)3 + 4 \underset{ \text{ 수소 }}{ H2 } }$$
수소화칼륨 + 물 $$\ce{ KH + \underset{ \text{ 물 }}{ H2O } → \underset{ \text{ 수산화칼륨 }}{ KOH }+ H2}$$
아세톤 연소식 $$\ce{ \underset{ \text{ 아세톤 }}{ CH3COCH3 } + 4O2 → 3 \underset{ \text{ 이산화탄소 }}{ CO2 } + 3 \underset{ \text{ 물 }}{ H2O } }$$
아세트산 + 과산화나트륨 $$\ce{2 \underset{ \text{ 아세트산 }}{ CH3COOH } + 2 \underset{ \text{ 과산화나트륨 }}{ Na2O2 } → 2CH3COONa + \underset{ \text{ 과산화수소 }}{ H2O2 } }$$
아세트산 연소식 $$\ce{ \underset{ \text{ 아세트산 }}{ CH3COOH } + 2 \underset{\text{산소}}{ O2 } → 2 \underset{\text{이산화탄소}}{ CO2 } + 2 \underset{ \text{ 물 }}{ H2O } }$$
아세트알데히드 산화식 $$\ce{ CH3CHO + 1/2O2 → CH3COOH}$$
아세틸렌 + 구리 $$\ce{ \underset{ \text{ 아세틸렌 }}{ C2H2 } + 2 \underset{ \text{ 구리 }}{ Cu } → Cu2C2 + H2}$$
아세틸렌 연소식 $$\ce{ 2 \underset{ \text{ 아세틸렌 }}{ C2H2 } + 5 \underset{ \text{ 산소 }}{ O2 } → 4 \underset{ \text{ 이산화탄소 }}{ CO2 } + 2H2O}$$
아연 + 염산 $$\ce{ Zn + 2 \underset{ \text{ 염산 }}{ HCl } → ZnCl2 + \underset{ \text{ 수소 }}{ H2 } }$$
아연 + 황산 $$\ce{Zn + \underset{ \text{ 황산 }}{ H2SO4 } → ZnSO4 + \underset{ \text{ 수소 }}{ H2 } }$$
아염소산나트륨 + 알루미늄 $$\ce{3NaClO2 + 4Al → 2Al2O3 + 3NaCl }$$
알루미늄 + 물 $$\ce{ 2Al + 6 \underset{ \text{ 물 }}{ H2O } → 2Al(OH)3 + 3H}$$
알루미늄 연소식 $$\ce{ 4Al + 3 \underset{\text{산소}}{ O2 } → 2Al2O3}$$
에탄올(에틸알코올) + 칼륨 $$\ce{ 2 \underset{ \text{ 에탄올 }}{ C2H5OH } + 2K → 2C2H5OK + H2}$$
에탄올(에틸알코올) 연소식 $$\ce{ \underset{ \text{ 에탄올 }}{ C2H5OH } + 3 \underset{\text{산소}}{ O2 } → 2 \underset{\text{이산화탄소}}{ CO2 } + 3 \underset{ \text{ 수소 }}{ H2 } }$$
에틸렌 산화식 $$\ce{ \underset{ \text{ 에틸렌 }}{ C2H4 } + PdCl2 + \underset{ \text{ 물 }}{ H2O } → CH3CHO + Pd + 2HCl}$$
염산 + 질산은 $$\ce{ \underset{ \text{ 염산 }}{ HCl } + \underset{ \text{ 질산은 }}{ AgNO3 } → HNO3 + AgCl}$$
염소산칼륨 + 황산 $$\ce{ 6 \underset{ \text{ 염소산칼륨 }}{ KClO3 } + 3 \underset{ \text{ 황산 }}{ H2SO4 } → 2 \underset{ \text{ 과염소산 }}{ HClO4 } + 3 \underset{ \text{ 황산칼륨 }}{ K2SO4 } + 4ClO2 + 2H2O}$$
염소산칼륨 400℃ 열분해식 $$\ce{ 2 \underset{ \text{ 염소산칼륨 }}{ KClO3 } → \underset{ \text{ 염화칼륨 }}{ KCl } + KClO4 + O2}$$
염소산칼륨 540~560℃ 열분해식 $$\ce{2 \underset{ \text{ 염소산칼륨 }}{ KClO3 } → 2 \underset{ \text{ 염화칼륨 }}{ KCl } + 3 \underset{\text{산소}}{ O2 } }$$
오황화린 + 물 $$\ce{ P2S5 + 8 \underset{ \text{ 물 }}{ H2O } → 2H3PO4 + 5H2S}$$
이산화망간 + 과산화수소 $$\ce{ \underset{\text{이산화망간}}{MnO2} + 2 \underset{ \text{ 과산화수소 }}{ H2O2 } → \underset{\text{이산화망간}}{MnO2} + 2 \underset{ \text{ 물 }}{ H2O } + O2}$$
이황화탄소 + 물 $$\ce{ CS2 + 2 \underset{ \text{ 물 }}{ H2O } → 2H2S + CO2}$$
이황화탄소 연소식 $$\ce{CS2 + 3 \underset{\text{산소}}{ O2 } → 2SO2 + \underset{ \text{ 이산화탄소 }}{ CO2 } }$$
인화알루미늄 + 물 $$\ce{ AlP + 3 \underset{ \text{ 물 }}{ H2O } → Al(OH)3 + PH3}$$
인화칼슘 + 물 $$\ce{Ca3P2 + 6 \underset{ \text{ 물 }}{ H2O } 3Ca(OH) → 2 + 2PH }$$
