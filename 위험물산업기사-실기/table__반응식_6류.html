반응 반응식
과염소산 가열분해 $$\ce{ \underset{ \text{ 과염소산 }}{ HClO4 } → \underset{ \text{ 염산 }}{ HCl } + \underset{ \text{ 산소 }}{ 2O2 } }$$
과산화수소 + 히드라진 (발화, 폭발) $$\ce{ \underset{ \text{ 과산화수소 }}{ 2H2O2 } + \underset{ \text{ 히드라진 }}{ N2H4 } → \underset{ \text{ 질소 }}{ N2 } + \underset{ \text{ 물 }}{ 4H2O } }$$
과산화수소 분해 $$\ce{ \underset{ \text{ 과산화수소 }}{ 2H2O2 } → \underset{ \text{ 물 }}{ 2H2O } + \underset{ \text{ 산소 }}{ O2 } }$$
과산화수소 안정제 $$\ce{ \underset{ \text{ 인산 }}{ H3PO4 } , \underset{ \text{ 요산 }}{ C5H4N4O3 } }$$
질산 분해 $$\ce{ \underset{ \text{ 질산 }}{ 4HNO3 } → \underset{ \text{ 물 }}{ 2H2O } + \underset{ \text{ 이산화질소 }}{ 4NO2 } + \underset{ \text{ 산소 }}{ O2 } }$$
