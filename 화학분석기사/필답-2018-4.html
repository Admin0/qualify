피로카테콜 바이올렛(Pyrocatechol violet)은 EDTA 적정 시 사용하는 지시약이다. 적정 과정이 다음과 같을 때, 각 물음에 답하시오. (5점)
  1. 금속용액에 완충용액과 지시약을 넣는다.
  2. EDTA를 과량 넣어 금속과 반응시킨 후, $\ce{Al3+}$로 역적정한다.
  3. 완충용액의 pH별 색깔은 다음과 같다.
  4. pH 6~7 적색, pH 7~8 노란색, pH 8~9 적색, pH 9~10 적자색
  5. 금속과 EDTA의 반응색은 파란색이다.
완충용액의 pH는?
pH 8~9
1) 반응 전, 착화물의 색인 푸른색과 명확히 구분되는 색을 가지되,
2) 조건형성상수가 가능한 큰 pH 위치를 찾아야 한다.
그러므로, pH 8~9 의 완충용액이 적당하다.
적정 전과 후의 색깔은?
파란색 → 노란색
역적정이므로 파란색에서 적정을 시작하여 적정이 끝나감에 따라 노란색으로 변한다.

다음은 전열원자화 장치에서 일어나는 원자화 단계에 대해서 서술 한 것이다. 빈 칸 안에 알맞은 단어를 채우시오.
시료를 낮은 온도에서 증발해서 ( 건조(drying) ) 시키고 전기적으로 가열된 흑연관 또는 흑연컵의 약간 높은 온도에서 ( 회화(ashing) ) 전류를 수백 A까지 빠르게 증가시켜 온도를 2000~3000도로 한다.
* 감도가 높다 - 이유는?
ⓐ 전체 시료가 짧은 시간에 원자화
ⓑ 원자가 빛 진로에 평균적으로 머무는 시간이 1초 이상
* 전열 원자화장치
→ 원자흡수법, 원자형광법에 사용
원자방출법에 사용하지 않음 → ICP의 시료증발장치로 사용
* 전열 원자화장치에서 원자화 단계
㉠ 건조: 수 μL 시료를 낮은 온도에서 증발
㉡ 회화: 전기적으로 가열된 흑연관 또는 흑연컵의 약간 높은 온도에서 회화
㉢ 원자화: 전류를 수백 A까지 빠르게 증가시켜 온도를 약 2000∼3000℃로 증가
→ 원자화는 수 ms ∼ 수 초 동안에 일어남
→ 원자화된 입자의 흡수 또는 형광은 가열 표면 바로 위 지역에서 측정
불꽃 전열 원자 흡수법에서 일어나는 방해는 크게 2가지이다. 스펙트럼 방해는 방해물질의 흡수선이 분석원소의 흡수선과 겹치거나 너무 가까워 단색화장치로 분리할 수 없을 때 생긴다.
