비열 $\pu{ 0.45 kcal/kg.^\circ C }$인 기름을 $\pu{ 1800 kg/hr }$로 열교환기에 흘려보내 온도를 $\pu{ 110^\circ C }$에서 $\pu{ 45^\circ C }$로 내렸다. 관 바깥에는 총전열계수가 $\pu{ 500 kcal/m2.hr.^\circ C }$인 수증기의 온도가 $\pu{ 20^\circ C }$에서 $\pu{ 50^\circ C }$로 변했다고 한다. 이 때 전열면적은 몇 $\pu{ m2 }$인가?
$ q = \dot{m}C_p\Delta T = \frac{\pu{ 1800 kg }}{\pu{ hr }} \times \frac{\pu{ 0.45 kcal }}{\pu{ kg * ^\circ C }} \times \pu{ (110 - 45)^\circ C } = \pu{ 52,650 kcal/hr } \\ \Delta T_1 = \pu{ (110 - 50)^\circ C } = \pu{ 60^\circ C }, \Delta T_2 = \pu{ (45 - 20)^\circ C } = \pu{ 25^\circ C }, \Longrightarrow \Delta T_{LM} = \dfrac{ \Delta T_1 - \Delta T_2 }{ \ln{ (\Delta T_1 / \Delta T_2) } } = \dfrac{ 60^\circ C - 25^\circ C }{ \ln{ (60^\circ C / 25^\circ C) } } = 39.9786^\circ C \\ q = UA \Delta T_{LM} = \pu{ 52,650 kcal/hr } = \pu{ 500 kcal/m2.hr.^\circ C } \times A \times \pu{ 39.9786^\circ C } \\ \therefore A = \pu{ 2.63 m2 } $

탱크의 압력
$ \rho = \pu{ 1840 kg/m3 }, D = \pu{ 0.076 m }, u = \pu{ 1 m/s }, z = \pu{ 10 m }, \eta = 0.7, f = \pu{ 3 m }, \\ \dot{m} = uA\rho = \left(\frac{\pu{ 1 m }}{\pu{ s }}\right) \times \left(\frac{\pu{ \pi }}{\pu{ 4 }} \times (\pu{0.076 m})^2\right) \times \left(\frac{\pu{ 1840 kg }}{\pu{ m3 }}\right) = \pu{ 8.3471 kg/s } \\ \begin{align*} W_\ce{P} & = \frac{\pu{ \Delta P }}{\pu{ \rho }} + \frac{\pu{ \Delta u2 }}{\pu{ 2 }} +g\Delta z + f \\ & = \left[0\right] + \left[\frac{\pu{ (1 m/s)2 }}{\pu{ 2 }}\right] + \left[ \frac{\pu{ 9.8 m/s2 }}{\pu{ 10 m }} \times \pu{10 m} \right] + \left[ \frac{\pu{ 9.8 m/s2 }}{\pu{ 10 m }} \times \pu{3 m} \right] \\ & = \pu{235336 kg/m.s2} \\ &= \pu{235336 N/m2} \end{align*} $
물필요 동력
$ \begin{align*} P &= \frac{ W_\ce{P}.\dot{m} }{ \eta } = \frac{ \pu{ 127.9 m2/s2 }\times \pu{ 8.3471 kg/s } }{0.7} \\ &= \pu{1525.13 J/s} \end{align*} $
